내부 클래스는 말 그대로 클래스 내에 선언된 클래스이다.
클래스 안에 또 다른 클래스를 선언한 것이다.

두 클래스의 멤버들 간 접근이 쉽다는 점,
외부에 불필요한 클래스를 감춤으로써 코드의 복잡성을 줄일 수 있다는 점이 장점이다.

## 내부 클래스의 종류

- 인스턴스 클래스 : 멤버변수 선언 위치, 인스턴스 멤버처럼
- 스태틱 클래스 : 멤버변수 선언 위치, static 멤버처럼
- 지역 클래스 : 메서드 내부 혹은 초기화 블럭 안에 선언
- 익명 클래스 : 일회용, 이름 없는 클래스
